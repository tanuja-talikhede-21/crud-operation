# 👥 User Management (CRUD) App

## 📌 Description

This is a simple **CRUD (Create, Read, Update, Delete)** application built using React.
It allows users to manage a list of users by adding, editing, and deleting data.

---

## 🚀 Features

* ➕ Add new user
* 📋 View user list
* ✏️ Edit user details
* ❌ Delete user
* 🔄 Real-time UI updates

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 📜 JavaScript (ES6)
* 🎨 CSS

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/tanuja-talikhede-21/crud-operation
```


2. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```
